# 🔍 competitor-analysis 使用指南

> **Skill 定位**：多维度竞品对比分析框架

---

## 🚀 使用方法

```
我的产品：连锁零售门店数字化平台
竞品：有赞零售、微盟智慧零售、慧策旺店通

请输出：
1. 每家的核心用户和市场定位
2. 核心功能对照表（详细到每个模块）
3. 每家的优缺点分析
4. 我们的差异化机会点
5. 跟进策略矩阵
```

---

## 💡 分析维度

| 维度 | 分析要点 |
|------|---------|
| 公司层面 | 融资情况、团队背景、战略重心 |
| 产品层面 | 功能覆盖度、用户体验、技术架构 |
| 商业层面 | 定价策略、客户分层、销售模式 |
| 运营层面 | 客户成功、培训体系、生态建设 |

---

## 📦 策略输出

| | 跟进成本低 | 跟进成本高 |
|--|-----------|-----------|
| 用户价值大 | ✅ 立刻抄 | ⏱️ 分阶段做 |
| 用户价值小 | 📌 观察 | ❌ 不做 |
