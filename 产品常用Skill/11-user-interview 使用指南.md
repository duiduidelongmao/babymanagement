# 👤 user-interview 使用指南

> **Skill 定位**：访谈提纲生成 + 纪要整理模板

---

## 🚀 阶段1：访谈前 - 生成提纲

```
我要访谈12位门店店长，主题是"数据报表的使用痛点"
请生成访谈提纲：
1. 开场破冰话术
2. 5个核心问题
3. 追问的技巧（每个问题怎么往下挖）
4. 结尾收尾和感谢话术
```

---

## 🚀 阶段2：访谈后 - 纪要整理

```
这里是5万字的12位店长的访谈原始记录

请帮我：
1. TOP 5痛点，每个标注N人提到
2. TOP 3亮点
3. 用户没说透的潜在需求
4. 不同类型店长的观点差异
5. 每个观点引用用户原话作为佐证
```

---

## 💡 用户访谈黄金法则

> **不要问用户想要什么功能，问用户遇到了什么问题**
>
> 不要问："你想要一个自动导出功能吗？"
>
> 要问："你现在导出数据一般是怎么操作的？上次操作时遇到了什么麻烦？"
