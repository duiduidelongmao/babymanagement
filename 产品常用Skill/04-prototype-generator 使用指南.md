# 🎨 prototype-generator 使用指南

> **Skill 定位**：100%还原设计规范生成可交互HTML原型

---

## 🎯 这个Skill解决什么问题

| 传统原型的问题 | 这个Skill的优势 |
|--------------|---------------|
| ❌ Axure画出来是假的，研发说"做不出来" | ✅ 生成真实HTML/CSS/JS，直接可运行 |
| ❌ 画高保真要3天，画完研发还要重新写 | ✅ 40分钟生成，研发可以直接复用代码 |
| ❌ 开评审会大家脑补交互效果 | ✅ 直接点击演示，没有信息损耗 |

---

## 🚀 使用方法

### 第一步：一句话描述你的页面

```
/skill prototype-generator

帮我生成一个门店数据看板页面：
1. 顶部4个数据卡片：营业额、客流、客单价、转化率
2. 中间是近7天趋势折线图
3. 下面是各门店排行表格
4. 用深蓝色科技感风格，加毛玻璃效果
```

### 第二步：迭代优化

**每轮只改一个点**：

```
第一轮：先看整体布局对不对
第二轮：调整配色和风格
第三轮：加上交互效果
第四轮：补全真实数据
```

## 💡 高级技巧

### 1. 控制精细度

| 想要的效果 | Prompt指令 |
|-----------|-----------|
| 快速看个大概 | "先不用太精细，先看整体布局" |
| 演示用高保真 | "加动画效果、过渡、悬停状态" |
| 给研发用的 | "代码结构清晰，加必要的注释" |

### 2. 设计系统注入

```
"严格按照Ant Design规范实现"
"参照Tailwind UI的风格"
"用我们公司的设计规范：主色#165DFF，圆角8px"
```
