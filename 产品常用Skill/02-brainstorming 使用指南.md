# 🧠 brainstorming 使用指南

> **Skill 定位**：战略顾问式追问，一次只问一个问题，苏格拉底式头脑风暴

---

## 🎯 这个Skill解决什么问题

| 普通头脑风暴的问题 | 这个Skill的优势 |
|----------------|---------------|
| ❌ 越聊越偏，最后什么结论都没有 | ✅ 步步紧逼，帮你挖到问题本质 |
| ❌ 大家都是拍脑袋，没有深度 | ✅ 层层递进，每个结论都有支撑 |
| ❌ 新手容易被资深人士带偏 | ✅ 苏格拉底式提问，谁都无法跳步 |

---

## 🚀 使用方法

### 第一步：抛出你的想法

```
/skill brainstorming

我想做一个给门店店长用的自动报表功能
```

### 第二步：迎接灵魂拷问

AI会**一次只问一个问题**，像顶级咨询顾问一样引导你：

**典型的追问路径：**

```
🤖 为什么店长需要自动报表？
→ 因为他们每天要手动导数据

🤖 为什么手动导数据是个问题？
→ 因为每天要花30分钟，很繁琐

🤖 这30分钟对店长来说很宝贵吗？
→ 非常宝贵，高峰期店里缺人

🤖 如果他们省了这30分钟，会去做什么？
→ 可以在前台接客，提升营业额

🤖 所以这个功能的本质不是省时间，而是帮门店多赚钱对吗？
→ 对！

🤖 那为什么不直接增加一个员工呢？
→ ...
```

## 💡 最佳实践

### ✅ 高级用法

1. **"我想不出来"是最好的答案**
   > 这恰恰是这个Skill最有价值的时刻，AI会帮你打开思路

2. **被问住了应该庆祝**
   > 说明你发现了自己之前的思维盲区

3. **随时可以回溯**
   > "回到上一个问题，我觉得刚才的回答不对"

### ❌ 典型的错误用法

- ❌ 试图反问AI："你觉得呢？"
- ❌ 不耐烦："别问了，直接给方案"
- ❌ 回答模棱两可："差不多吧"、"都行"

## 🎯 核心理念

> **苏格拉底教学法的精髓**：
> 
> 我不告诉你答案，我只负责提问，
> 
> 因为你自己心里其实已经有答案，
> 
> 只是你还没意识到它在那里。

这个Skill不会给你任何答案，但会让你在10轮对话后，自己找到那个最清晰的答案。
