# ⚠️ risk-assessment 使用指南

> **Skill 定位**：项目风险识别与应对矩阵

---

## 🚀 使用方法

```
项目：新版权限系统重构
时间：8周
团队：前端2人，后端2人，测试1人

请帮我识别：
1. 技术风险
2. 业务风险
3. 项目管理风险
4. 外部依赖风险
5. 每个风险的应对方案、责任人、触发条件
```

---
