# 📊 ab-test-setup 使用指南

> **Skill 定位**：样本量计算、指标设计、统计显著性解读

---

## 🎯 这个Skill解决什么问题

| 普通人做A/B测试 | 用这个Skill |
|--------------|------------|
| ❌ 跑了3天看到有提升就全量 | ✅ 告诉你样本量够不够，有没有统计显著性 |
| ❌ 只看转化率一个指标 | ✅ 帮你设计完整的指标矩阵 |
| ❌ 不知道为什么数据波动 | ✅ 教你怎么排除新奇效应等干扰 |

---

## 🚀 使用方法

```
/skill ab-test-setup

实验：把首页的立即注册按钮从蓝色改成红色
目前基准转化率：5%
希望提升：至少10%
每天流量：10000人

请输出：
1. 需要多大样本量
2. 需要跑多少天
3. 核心指标、护栏指标、探索指标分别是什么
4. 怎么判断实验成功还是失败
5. 常见的坑有哪些
```

---

## 💡 核心价值

### 再也不要犯这些经典错误：

1. **过早看结果** → Skill会告诉你"第3天之前不要去看数据"
2. **多重检验偏差** → 同时看10个指标，总有一个"统计显著"
3. **新奇效应** → 用户前3天觉得新鲜点的多，一周后恢复正常
4. **季节因素** → 周末和工作日的数据完全不一样

---

## 📦 完整输出

```
📋 A/B测试方案
├── 📊 实验参数
│   ├── 95%置信度下需要样本：每组31,000人
│   ├── 需要实验天数：7天
│   └── 分配比例：50% / 50%
├── 🎯 指标体系
│   ├── 核心指标：按钮点击率
│   ├── 护栏指标：页面跳出率
│   └── 探索指标：后续注册转化率
├── ✅ 成功标准
│   └── p值 < 0.05 且 提升 > 5%
└── ⚠️ 注意事项
    ├── 前3天数据有新奇效应，不要看
    ├── 不要每天看数据决定要不要停
    └── 分新老用户看细分效果
```
