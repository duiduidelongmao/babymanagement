# 📋 user-story 使用指南

> **Skill 定位**：INVEST原则生成标准用户故事

---

## 🎯 INVEST 黄金原则

| 原则 | 说明 |
|------|------|
| 🇮 Independent | 独立，不依赖其他用户故事 |
| 🇳 Negociable | 可协商，不是合同 |
| 🇻 Valuable | 对用户有明确价值 |
| 🇪 Estimable | 可估时 |
| 🇸 Small | 小，一个Sprint能做完 |
| 🇹 Testable | 可测试 |

---

## 🚀 使用方法

```
需求：门店报表自动推送功能

请按INVEST原则拆分成用户故事，格式：
> 作为 <什么用户>，我想要 <做什么>，以便于 <获得什么价值>

每个故事包含：
1. 优先级
2. 故事点估算
3. 验收标准要点
4. 是否适合放在MVP
```
