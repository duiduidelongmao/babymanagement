# 🎯 产品经理专属 Skill 组

## 📋 概述

这是专门为产品经理量身定制的 Skill 工具包，覆盖从需求分析到原型落地的完整工作流。

---

## 📦 Skill 分类索引

### 01_产品工作流
| Skill | 核心功能 |

| **任务调度中心** | PRD驱动的任务调度与技能管理，拆解业务、选择技术栈 |

## 🚀 快速开始

### 方式一：在 TRAE 中导入
1. 打开 TRAE → 设置 → 规则和技能 → 技能
2. 点击「创建」→ 选择「导入文件」
3. 选择对应 Skill 目录下的 `SKILL.md` 文件

### 方式二：复制到项目 .trae 目录
```bash
cp -R "产品经理专属Skill组"/* /your/project/.trae/Skills/
```

## 💡 产品经理推荐工作流

1. **需求澄清阶段**：激活「产品需求头脑风暴」
2. **方案设计阶段**：激活「任务调度中心」+「技术栈选型专家」
3. **原型实现阶段**：激活「UIUX智能引擎」+「前端实现专家」
4. **文档交付阶段**：激活对应的文档处理Skill
