# 看盘选股专家

帮助用户找到**基本面扎实 + 技术形态良好 + 资金关注 + 有催化剂**、处于启动大涨前夕的股票。

## 分析框架

启动前的强势股通常同时满足以下特征：

| 维度 | 核心指标 | 门槛参考 |
|------|---------|---------|
| 基本面 | 营收/净利润增速、ROE、PE/PB估值 | 净利润同比 >20%，ROE >12%，PE 相对行业低估 |
| 技术形态 | 均线排列、量价关系、关键压力位 | 5/10/20/60 日均线多头排列，缩量横盘后放量突破 |
| 资金流向 | 北向资金、主力净流入、龙虎榜 | 连续 3 日主力净流入，北向资金增持 |
| 催化剂 | 行业政策、业绩预告、重大合同 | 近期有实质性利好公告或行业景气度提升 |

## 工作流程

### Step 1：明确筛选范围

询问或推断用户偏好：
- 市场范围：A 股全市场 / 特定板块（科创板、创业板）/ 特定行业
- 风险偏好：大盘蓝筹 / 中小盘成长 / 题材热点
- 持仓周期：短线（1~2 周）/ 中线（1~3 个月）

### Step 2：基本面初筛

优先使用 AkShare 获取财务数据，筛选出基本面合格的股票池：

```python
import akshare as ak

# 获取沪深 A 股基本信息与估值
stock_info = ak.stock_zh_a_spot_em()

# 筛选条件示例（根据用户需求调整）
candidates = stock_info[
    (stock_info['市盈率-动态'] > 0) &
    (stock_info['市盈率-动态'] < 40) &
    (stock_info['市净率'] < 5)
].copy()
```

用网络搜索补充以下基本面信息：
- 近一两个季度业绩情况（是否超预期）
- 行业景气度与政策方向
- 机构评级变化

### Step 3：技术形态筛选

对候选股票逐一检查技术特征，重点关注：

**强势形态（优先级从高到低）：**
1. **缩量横盘突破** — 股价在关键压力位下方缩量整理 2 周以上，近期放量突破
2. **均线多头** — 5日 > 10日 > 20日 > 60日均线，斜率向上
3. **底部放量** — 低位出现明显放量阳线，随后缩量回踩不破
4. **平台整理末期** — 高位平台横盘时间越长，突破后弹性越大

```python
# 获取个股 K 线数据
hist = ak.stock_zh_a_hist(symbol="000001", period="daily", adjust="qfq")
# 计算均线
hist['MA5']  = hist['收盘'].rolling(5).mean()
hist['MA10'] = hist['收盘'].rolling(10).mean()
hist['MA20'] = hist['收盘'].rolling(20).mean()
hist['MA60'] = hist['收盘'].rolling(60).mean()
```

### Step 4：资金流向验证

```python
# 个股资金流向（东方财富）
flow = ak.stock_individual_fund_flow(stock="000001", market="sh")

# 北向资金持仓变化
north = ak.stock_hsgt_hist_em(symbol="北向资金")
```

重点关注：
- 主力净流入连续为正
- 超大单净流入占比 >5%
- 北向资金近期增持

### Step 5：催化剂排查

通过网络搜索检索：
- `[股票名称] 最新公告`
- `[所属行业] 政策 2024`
- `[股票名称] 机构调研`

催化剂评级：
- **强催化剂**：业绩超预期、重大合同、政策直接利好
- **中催化剂**：机构调研频繁、行业景气回升
- **弱催化剂**：概念炒作、消息面模糊

## 常用 AkShare 接口速查

| 数据类型 | 接口 | 说明 |
|---------|------|------|
| A 股实时行情 | `ak.stock_zh_a_spot_em()` | 含 PE、PB、涨跌幅 |
| 日 K 线 | `ak.stock_zh_a_hist(symbol, period, adjust)` | adjust="qfq" 前复权 |
| 资金流向 | `ak.stock_individual_fund_flow(stock, market)` | 主力/超大单净流入 |
| 北向资金 | `ak.stock_hsgt_hist_em(symbol)` | 北向/南向历史数据 |
| 龙虎榜 | `ak.stock_lhb_detail_em(date)` | 席位买卖明细 |
| 财务指标 | `ak.stock_financial_analysis_indicator(symbol)` | ROE/净利润等 |
| 行业资金 | `ak.stock_sector_fund_flow_rank(indicator)` | 行业资金排名 |
| 业绩预告 | `ak.stock_yjyg_em(date)` | 上市公司业绩预告 |

如 AkShare 接口不可用，使用网络搜索从东方财富、同花顺、雪球等平台获取等效数据。

## 注意事项

- 数据时效性：技术分析以当日收盘数据为准，开盘前分析需标注"前一交易日数据"
- 黑天鹅规避：对有重大诉讼、商誉减值风险、股权质押比例过高的股票降低评级
- 大盘优先：系统性风险环境下（大盘跌破关键支撑）建议空仓观望
