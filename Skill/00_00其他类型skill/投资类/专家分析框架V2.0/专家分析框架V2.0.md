# 专家分析框架 V2.0

> V1.1 深度产业研判 + V1.0 技术择时与风险扫描 合并升级

## 模型总览

**产业-宏观-估值-技术-风险五维动态评估决策模型**

```
┌─────────────────────────────────────────────────────────┐
│             宏观状态检测器（前置模块）                       │
│       周度更新 · 十大核心指标 · 四大辅助指标                  │
│             扩张 / 中性 / 收缩                             │
└───────────────┬─────────────────────────────────────────┘
                ▼
┌───────────────┼─────────────────────────────────────────┐
│  核心三维（75%）           │  辅助二维（25%）              │
│                           │                              │
│  宏观维度(20%)            │  技术走势(15%)               │
│   ·美林时钟               │   ·趋势方向                   │
│   ·信用周期               │   ·量价关系                   │
│   ·流动性                 │   ·技术形态                   │
│   ·通胀趋势               │   ·指标信号                   │
│   ·地缘政治 [NEW]         │   ·筹码分布                   │
│   ·供应链安全 [NEW]       │                              │
│                           │                              │
│  产业维度(30%)            │  风险评估(10%)               │
│   ·行业分类适配            │   ·市场风险(beta/波动率)      │
│   ·S曲线三层穿透           │   ·信用风险                   │
│   ·三因子漏斗              │   ·流动性风险                 │
│                           │   ·事件风险                   │
│  估值维度(15%)            │   ·估值风险                   │
│   ·Gartner情绪            │                              │
│   ·市值天花板              │                              │
│   ·PEG验证                │                              │
│   ·估值切换信号 [NEW]      │                              │
└───────────────┴─────────────────────────────────────────┘
                ▼
      五维共振定级 → 风险调整 → 仓位决策
```

**核心逻辑**：
> 前置过滤（宏观检测器）→ 核心三维共振（产业+宏观+估值）→ 辅助二维调整（技术择时+风险约束）→ 仓位决策

## 第一步：宏观评分（权重 20%）

| 评估项目 | 评分依据 | 数据来源 |
|---------|---------|---------|
| 美林时钟象限 | 根据PMI与CPI定位当前经济象限（复苏/过热/滞胀/衰退），复苏象限最优 | PMI: 统计局; CPI: 统计局 |
| 信用周期 | 产业债信用利差水平，利差收窄=信用宽裕，利差走阔=信用收紧 | 信用利差: Wind/东方财富 |
| 流动性环境 | 中国LPR/MLF走势 + 美联储利率决议预期，双宽最优 | LPR: 央行; Fed: CME FedWatch |
| 通胀趋势 | CPI温和回升 + PPI-CPI剪刀差转正 = 企业盈利改善信号 | CPI/PPI: 统计局 |
| 地缘政治风险 [NEW] | 地缘冲突是否催生国产替代机遇；冲突升级但受益国产替代可加分 | 新闻/政策文件 |
| 供应链安全 [NEW] | 关键环节对外依存度；国产替代政策加码程度 | 行业报告/政策文件 |

**评分输出**：宏观综合评分 = XX/100

**仓位区间映射**：
- ≥ 80分 → 高仓位进攻区间
- 60~79分 → 标准配置区间
- 40~59分 → 轻仓防御区间
- < 40分 → 观望/空仓

## 第二步：产业阶段评分（权重 30%）

### 2.1 行业分类定位

**驱动力类型**（决定后续三因子权重分配）：

| 类型 | 标识 | 典型行业 | 核心驱动力 |
|------|------|---------|-----------|
| A | Capex驱动型 | 半导体设备、光伏 | 下游资本开支周期 |
| B | 国产替代/安全驱动型 | 特种芯片、信创 | 自主可控紧迫度 + 技术壁垒 |
| C | 政策驱动型 | 新能源、环保 | 政策补贴/强制性标准 |
| D | 消费驱动型 | 白酒、医药 | 终端消费需求 |

**产业阶段**（S曲线定位）：

| 阶段 | 渗透率区间 | 投资含义 |
|------|-----------|---------|
| 导入期 | <5% | 高不确定性，主题投资 |
| 甜蜜区 | 5%~30% | 业绩加速兑现，最佳配置窗口 |
| 成熟期 | >30% | 增速放缓，看份额集中/出海 |

### 2.3 三因子漏斗逐层评估

## 第五步：风险评估（权重 10%）

> 风险评估为"减速带"机制：高分不直接提升总评，但低分会显著压制最终评级。

### 5.1 五类风险评估

| 风险类型 | 核心指标 | 低风险(8~10分) | 中风险(4~7分) | 高风险(1~3分) |
|---------|---------|---------------|--------------|--------------|
| 市场风险 | Beta系数、年化波动率 | Beta<0.8，波动率<行业均值 | Beta 0.8~1.2 | Beta>1.5，波动率>行业2倍 |
| 信用风险 | 债务评级、违约概率 | AAA/AA+，无违约风险 | AA/AA- | A及以下，有违约疑虑 |
| 流动性风险 | 日均成交额、买卖价差 | 日均成交>5亿，价差<0.1% | 日均1~5亿 | 日均<1亿，价差>0.5% |
| 事件风险 | 诉讼/减持/监管/质押 | 无重大事件 | 中等影响事件 | 重大诉讼/大额减持/高质押 |
| 估值风险 | 估值偏离度 | 低于行业均值 | 接近行业均值 | 高于行业均值50%以上 |

### 5.2 风险压力测试

对标的进行极端情景测试：

| 情景 | 测试方法 | 关注指标 |
|------|---------|---------|
| 大盘暴跌10% | Beta×大盘跌幅估算最大回撤 | 最大亏损是否可承受 |
| 行业政策突变 | 假设核心政策取消/收紧 | 营收影响幅度 |
| 核心客户流失 | 假设前3大客户流失 | 营收/利润下降比例 |
| 利率上行200bp | 估值折现率上升对PE的影响 | 目标价下调幅度 |

### 5.3 风险评分与调整系数

**风险综合评分** = 五类风险加权平均（市场30% + 信用15% + 流动性15% + 事件25% + 估值15%）

> 风险评分采用反向计分：10分=最低风险，1分=最高风险

**风险调整系数**（乘以前四维加权总分）：

| 风险综合评分 | 调整系数 | 含义 |
|-------------|---------|------|
| 8~10分 | ×1.0 | 风险可控，不影响仓位 |
| 6~7分 | ×0.85 | 风险中等，仓位打85折 |
| 4~5分 | ×0.65 | 风险偏高，仓位大幅缩减 |
| 1~3分 | ×0.4 | 风险极高，仅保留观察仓 |

## 最终决策：五维共振定级

### 总分计算

```
加权总分 = 宏观评分×0.20 + 产业评分×0.30 + 估值评分×0.15 + 技术评分×0.15 + 风险评分×0.10
调整总分 = 加权总分 × 风险调整系数
```

### 核心三维共振等级

先以核心三维（宏观+产业+估值，合计65%权重）判断共振等级：

| 等级 | 条件 | 含义 |
|------|------|------|
| ★★★ | 核心三维均≥80且无维度<60 | 强共振，高确定性机会 |
| ★★☆ | 两维≥80 + 一维≥60 | 较强共振，可重仓 |
| ★★ | 一维≥80 + 两维≥60 | 一般共振，标准配置 |
| ★ | 无维度≥80或任一维度<50 | 弱共振，轻仓或观望 |

### 安全溢价调整

若分析标的属于 **国产替代/安全驱动型(B类)** 且安全溢价因子得分 ≥8/10，共振等级上调半级。

### 技术走势确认

| 技术评分 | 对共振等级的影响 |
|---------|----------------|
| ≥80 | 共振等级维持或上调半级（技术确认） |
| 60~79 | 不影响共振等级 |
| 40~59 | 共振等级下调半级（技术面偏弱） |
| <40 | 共振等级下调一级（技术面否定） |

### 仓位决策

| 决策 | 条件 | 建议仓位 |
|------|------|---------|
| 重仓买入 | ★★★ + 技术≥60 + 风险调整系数≥0.85 | 15%~25% |
| 标准配置 | ★★☆ 或 ★★ + 技术≥60 | 8%~15% |
| 轻仓试探 | ★★ + 技术50~59 或 ★ + 估值切换信号≥3/4 | 3%~8% |
| 减仓/退出 | ★ 或 宏观收缩信号≥5 或 技术<40 | 0% |

## 使用指引

### 触发场景

- 用户要求系统分析某只股票
- 用户询问某行业是否值得配置
- 用户要求评估买入时机与仓位
- 用户需要对比多只股票的投资价值
- 用户需要技术择时确认买卖点
- 用户要求评估投资风险与压力测试

### 执行要点

1. **先宏观后个股**：必须先完成宏观检测器，收缩环境下降低所有个股评级
2. **行业分类决定权重**：务必先判断驱动力类型(A/B/C/D)，再套用对应三因子权重
3. **技术走势是确认项**：核心三维定方向，技术走势定时机；技术面弱势可否决基本面看好
4. **风险是减速带**：风险评分不直接加分，但低分会通过调整系数大幅压制最终评级
5. **估值切换信号是加分项**：4项信号满足3项以上，可在共振等级上额外加分
6. **风险提示必须包含**：每次输出必须包含"主要风险"和"压力测试"栏位
7. **数据时效性**：每次输出必须检查数据时效性，不可输出滞后数据；技术分析以当日收盘数据为准
