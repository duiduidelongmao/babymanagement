# 专家分析框架

## 一、框架概述

本框架采用**五维评估体系**，从宏观到微观、从定性到定量，全面评估投资标的的投资价值与风险等级，为投资决策提供科学依据。

---

## 二、五维评估体系

### 维度一：宏观环境分析

| 指标类型 | 核心指标 | 评估标准 |
|---------|---------|---------|
| 经济周期 | GDP增速、PMI指数 | 扩张期/收缩期判断 |
| 货币政策 | 利率、M2增速、信贷规模 | 宽松/中性/紧缩 |
| 财政政策 | 财政支出、税收政策 | 积极/稳健/保守 |
| 汇率环境 | 人民币汇率走势 | 升值/贬值趋势 |
| 市场情绪 | 风险偏好指数、波动率 | 高/中/低风险偏好 |

### 维度二：行业景气分析

| 指标类型 | 核心指标 | 评估标准 |
|---------|---------|---------|
| 行业生命周期 | 渗透率、增长率 | 导入期/成长期/成熟期/衰退期 |
| 供需格局 | 产能利用率、库存水平 | 供不应求/供需平衡/供过于求 |
| 竞争格局 | CR4/CR10、行业集中度 | 垄断/寡头/竞争/完全竞争 |
| 政策影响 | 行业政策支持力度 | 强支持/中性/强监管 |
| 技术变革 | 技术创新速度、替代威胁 | 颠覆性/渐进性/无变革 |

### 维度三：公司基本面分析

| 指标类型 | 核心指标 | 评估标准 |
|---------|---------|---------|
| 盈利能力 | ROE、毛利率、净利率 | 高于行业平均+持续稳定 |
| 成长能力 | 营收增速、净利润增速 | 连续3年>行业平均 |
| 财务健康 | 资产负债率、现金流 | 负债合理、现金流充沛 |
| 管理质量 | 管理层能力、治理结构 | 优秀/良好/一般/较差 |
| 估值水平 | PE、PB、EV/EBITDA | 相对行业合理/低估/高估 |

### 维度四：技术走势分析

| 指标类型 | 核心指标 | 评估标准 |
|---------|---------|---------|
| 趋势方向 | 均线系统、趋势线 | 多头排列/空头排列/震荡 |
| 量价关系 | 成交量、换手率 | 价涨量增/价涨量缩/价跌量增 |
| 技术形态 | 突破、回调、反转信号 | 有效突破/假突破/失败形态 |
| 指标信号 | MACD、KDJ、RSI | 金叉/死叉、超买/超卖 |
| 筹码分布 | 持仓成本、集中度 | 筹码集中/分散 |

### 维度五：风险评估

| 风险类型 | 核心指标 | 评估标准 |
|---------|---------|---------|
| 市场风险 | beta系数、波动率 | 高/中/低风险 |
| 信用风险 | 违约概率、评级 | AAA/AA/A/BBB及以下 |
| 流动性风险 | 日均成交额、买卖价差 | 流动性充裕/一般/不足 |
| 事件风险 | 政策变动、诉讼、减持 | 重大/中等/轻微 |
| 估值风险 | 估值偏离度 | 高估/合理/低估 |

## 三、评估流程

### Step 1：初步筛选

- 确定投资目标与约束条件
- 设定初步筛选标准
- 建立候选标的池

### Step 2：维度分析

```python
# 评估框架示例代码
def evaluate_stock(symbol):
    scores = {
        'macro': analyze_macro(),      # 宏观环境评分 0-100
        'industry': analyze_industry(), # 行业景气评分 0-100
        'fundamental': analyze_fundamental(symbol), # 基本面评分 0-100
        'technical': analyze_technical(symbol),     # 技术面评分 0-100
        'risk': analyze_risk(symbol)                # 风险评分 0-100（反向）
    }
    
    # 综合评分（权重可调整）
    weights = {'macro': 0.15, 'industry': 0.20, 'fundamental': 0.30, 'technical': 0.20, 'risk': 0.15}
    total_score = sum(scores[k] * weights[k] for k in weights)
    return total_score
```

### Step 3：综合评级

| 综合评分 | 评级 | 投资建议 |
|---------|------|---------|
| 90-100 | A+ | 强烈推荐买入 |
| 80-89 | A | 推荐买入 |
| 70-79 | B+ | 谨慎买入 |
| 60-69 | B | 观望 |
| 50-59 | C+ | 谨慎卖出 |
| <50 | C | 卖出 |

### Step 4：策略建议

根据评级结果，输出：
- 买入/持有/卖出建议
- 目标价位区间
- 止损位设定
- 仓位建议
- 持有周期

## 六、注意事项

1. **动态更新**：市场环境变化快速，评估结果需定期更新
2. **定性判断**：部分指标需结合专家经验判断，非完全量化
3. **黑天鹅事件**：极端事件可能影响评估准确性
4. **组合视角**：单一标的评估需结合整体投资组合
5. **风险控制**：任何投资建议需配合严格的风险控制措施
