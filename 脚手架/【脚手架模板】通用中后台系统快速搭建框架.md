## 🚀 Step 0：工程初始化 - 30秒

### 执行命令

```bash
# 1. 创建 Vue3 + TypeScript 项目
npm create vite@latest . -- --template vue-ts

# 2. 安装核心依赖（版本固定，不要用最新版）
npm install vue@3.4.21 vue-router@4.3.0 pinia@2.1.7
npm install element-plus@2.6.1 @element-plus/icons-vue@2.3.1
npm install axios@1.6.7 dayjs@1.11.10 lodash-es@4.17.21

# 3. 开发依赖
npm install -D sass@1.72.0 typescript@5.4.2
npm install -D unplugin-auto-import@0.17.5 unplugin-vue-components@0.26.0
```

## ✅ Step 1：技术栈固定 - 不用思考直接抄

| 技术 | 版本号 | **为什么选它，为什么版本不能乱升** |
|------|--------|--------------------------------|
| **Vue** | 3.4.21 | ✅ 最稳定的版本，3.5+ 有兼容性问题 |
| **Vite** | 5.1.6 | ✅ 不要用 5.2+，Element Plus 有样式警告 |
| **TypeScript** | 5.4.2 | ✅ 超过 5.5 类型检查会变慢 |
| **Element Plus** | 2.6.1 | ✅ 2.7+ 弹窗 z-index 有 Bug |
| **Pinia** | 2.1.7 | ✅ Vue 官方推荐，废弃 Vuex |
| **Sass** | 1.72.0 | ✅ 不要用 1.77+，legacy API 警告刷屏 |
| **unplugin** | 0.17.x | ✅ 自动导入，永远不要写 import Element Plus |

> 💡 **黄金法则：** 以上版本号，一字不差！不要 `npm install xxx@latest`

## 🎨 Step 2：三栏布局 - 像素级对齐标准

### 黄金尺寸（抄就对了）

| 区域 | 宽度 | 背景色 | 内边距 | 边框 |
|------|------|--------|--------|------|
| **顶部导航** | 100% | white / #111827 | `0 24px` | border-bottom 1px |
| **左侧菜单** | 240px | white / #1f2937 | 0 | border-right 1px |
| **右侧主体** | calc(100% - 240px) | #f9fafb / #0f172a | `24px` | 无 |

### MainLayout.vue 标准模板

```vue
<template>
  <div class="main-layout">
    <!-- 顶部导航：高度 64px 是黄金标准 -->
    <header class="layout-header" style="height: 64px">
      <slot name="header" />
    </header>
    
    <div class="layout-body">
      <!-- 左侧菜单：宽度 240px 不要改 -->
      <aside class="layout-sidebar" style="width: 240px">
        <slot name="sidebar" />
      </aside>
      
      <!-- 右侧内容：内边距统一 24px -->
      <main class="layout-content" style="padding: 24px">
        <router-view />
      </main>
    </div>
  </div>
</template>
```

### ✅ 像素级对齐验收

- [ ] 头像与文字 **垂直居中**，差 1px = 不合格
- [ ] 所有卡片 `border-radius: 12px`，不要 8px 也不要 16px
- [ ] 卡片内边距统一 `24px`
- [ ] 按钮高度统一 `36px`

## 🔐 Step 4：权限系统 - RBAC 标准模型

### 不要创新，直接用这套

#### 1. 用户登录 Store

```typescript
// store/user.ts
export const useUserStore = defineStore('user', {
  state: () => ({
    token: localStorage.getItem('token') || '',
    userInfo: null,
    roles: [],
    permissions: []
  }),
  
  actions: {
    // 登录
    async login(form) {
      this.token = 'mock-token-' + Date.now()
      localStorage.setItem('token', this.token)
    },
    
    // 退出
    logout() {
      this.token = ''
      localStorage.removeItem('token')
      router.push('/login')
    }
  }
})
```

#### 2. 路由守卫标准模板

```typescript
router.beforeEach((to, from, next) => {
  const token = localStorage.getItem('token')
  
  // 1. 未登录只能看登录页
  if (!token && to.path !== '/login') {
    return next('/login')
  }
  
  // 2. 已登录不能看登录页
  if (token && to.path === '/login') {
    return next('/main/workspace')
  }
  
  next()
})
```

### ✅ 权限系统 CheckList

- [ ] 路由守卫，未登录自动跳登录
- [ ] 登录成功跳工作台
- [ ] 退出登录清除所有状态
- [ ] F5 刷新不丢失登录

## ⚙️ Step 7：Vite 最优配置

### 直接复制粘贴 vite.config.ts

```typescript
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'
import { resolve } from 'path'

export default defineConfig({
  plugins: [
    vue(),
    
    // ✅ 魔法：再也不用 import { ref, reactive } 了
    AutoImport({
      imports: ['vue', 'vue-router', 'pinia'],
      resolvers: [ElementPlusResolver()],
      dts: 'src/auto-imports.d.ts',
      eslintrc: { enabled: true }
    }),
    
    // ✅ 魔法：Element Plus 组件自动导入
    Components({
      resolvers: [ElementPlusResolver()],
      dts: 'src/components.d.ts'
    })
  ],
  
  resolve: {
    alias: {
      '@': resolve(__dirname, 'src')
    }
  },
  
  server: {
    port: 2008,           // ✅ 端口号固定
    host: true,           // ✅ 允许局域网访问
    open: true            // ✅ 自动打开浏览器
  }
})
```
